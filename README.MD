<h1>Kisoro Shop</h1>

<h2>Getting Started</h2>

To avoid typing `npm run` repeatedly, check out [ALIAS](docs/ALIAS.MD)

----------

1. Develop
```
npm i
npm start
```

2. Test
```
npm run test
```

3. Deploy
```
nano .firebaserc -> add your firebase project name here
npm run fireup
```
