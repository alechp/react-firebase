<h2>Getting Started</h2>

To avoid typing `npm run` repeatedly, check out [ALIAS](docs/ALIAS.MD)

----------

1. Develop
```

npm start
```

2. Test
```
npm run test
```

3. Deploy
```
npm run fireup
```
